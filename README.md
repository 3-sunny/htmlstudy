## markdown和HTML对比  
- 脚注  
-- 创建脚注格式类似这样 [^RUNOOB]  
[^RUNOOB]: 菜鸟教程 -- 学的不仅是技术，更是梦想！！！  
-- <!-- 这是一个注释，不会显示在页面上 -->  
- 分割线  
-- ***  
-- <hr/>  
- 文本格式化  
-- *斜体*    —斜体—    **粗体**    ***粗斜体***  
-- <b>加粗</b> <i>斜体</i> <strong>加粗</strong> <em>斜体</em>  
-- <big>放大</big> <small>缩小</small> <sub>下标</sub><sup>上标</sup>
